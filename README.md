What Is This?

How to Deploy to GitHub Pages
Option 1: Fork & Enable Pages (Recommended)
Fork this repo → click the Fork button at the top right
Go to Settings → Pages
Under "Source", select Deploy from a branch
Choose main branch, / (root) folder
Click Save
Your gallery will be live at https://your-username.github.io/ecommerce-intelligence-cards/
Option 2: Clone & Deploy
bash


git clone https://github.com/your-username/ecommerce-intelligence-cards.git
cd ecommerce-intelligence-cards
# Edit index.html if needed
git add .
git commit -m "Deploy gallery"
git push origin main
Then enable GitHub Pages in Settings as above.
